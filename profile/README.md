### Supported protocols

`VLESS` `Reality` `VMess` `Trojan` `Shadowsocks` `Hysteria2` `SOCKS5` `WireGuard`

### Key features

- **Subscriptions** — add servers via URL, QR, or share link with auto-refresh
- **Routing profiles** — split traffic by domain & IP: proxy, direct, or block
- **Full xray JSON import** — observatory, balancers, custom geo files
- **TLS fragmentation** — fragment ClientHello to bypass DPI
- **Siri Shortcuts** — connect, disconnect, toggle, and check status via voice
- **Customization** — 16 app icons, multiple themes, system tray with quick settings
- **Privacy** — zero data collection, everything stays on-device
